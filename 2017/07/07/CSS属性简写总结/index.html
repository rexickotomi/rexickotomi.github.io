CSS属性简写总结

Share

简写css属性时，如果你没有设定某些必要的属性值，这些属性值会使用缺省值，从而导致与预料的显示效果不同，这点要谨记。
此外，简写CSS属性时一定按照规定的顺序书写

如在一个html文件中，要注意css的引用顺序，如在引用的前一个css中，为body设置了background-color值，并且在引用的后一个css中也为body设置background的其他属性时，
如果使用了简写属性，就会导致起初的background-color值失效，而使用background-color的缺省值。

反之，如果引用的顺序相反，则不会发生这种情况，background-color值会覆盖缺省值。
